Loadings & Penalties
====================

Full-Time/Part-Time Penalties:
- Saturday: 125% (1.25x base rate)
- Sunday: 150% (1.5x base rate)
- Public Holiday: 225% (2.25x base rate)
- Evening (7pm+): 115% (1.15x base rate)
- Early Morning (before 7am): 115% (1.15x base rate)
- Late Night (after midnight): 130% (1.30x base rate)

Casual Penalties (add to casual loading):
- Casual Loading: 25% base (all shifts)
- Saturday: 150% (1.5x base rate total)
- Sunday: 175% (1.75x base rate total)
- Public Holiday: 250% (2.5x base rate total)

Overtime:
- First 2 hours over 38/week: 150% (1.5x base rate)
- After 2 hours over 38/week: 200% (2.0x base rate)

Other:
- Superannuation: 11.5% of ordinary time earnings
- Casual Loading: 25% of base hourly rate
- Junior Rates: Age-based percentage (16yo: ~51%, 17yo: ~65%, 18yo: ~80%)

Usage Example
=============

from rosteriq.award_engine import AwardEngine, EmploymentType
from datetime import date, time

engine = AwardEngine(award_year=2025)

# Calculate single shift cost
cost = engine.calculate_shift_cost(
    employee_id="EMP001",
    award_level=1,
    employment_type=EmploymentType.FULL_TIME,
    shift_date=date(2026, 3, 15),
    start_time=time(9, 0),
    end_time=time(17, 0),
    is_public_holiday=False
)

print(f"Gross Pay: ${cost.gross_pay}")
print(f"Super: ${cost.super_contribution}")
print(f"Employer Cost: ${cost.total_cost_to_employer}")

# Calculate weekly costs
weekly = engine.calculate_weekly_cost(
    employee_id="EMP001",
    award_level=1,
    employment_type=EmploymentType.FULL_TIME,
    shifts=[
        (date(2026, 3, 16), time(9, 0), time(17, 0)),
        (date(2026, 3, 17), time(9, 0), time(17, 0)),
        # ... more shifts
    ]
)

# Calculate roster costs
roster = engine.calculate_roster_cost(roster_data)
print(f"Total Employer Cost: ${roster.total_employer_cost}")
print(f"Total Superannuation: ${roster.total_super}")

# Check compliance
warnings = engine.check_compliance(
