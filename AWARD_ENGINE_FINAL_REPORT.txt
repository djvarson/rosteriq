USAGE EXAMPLES
==============

Basic Usage:
```python
from rosteriq.award_engine import AwardEngine, EmploymentType
from datetime import date, time

engine = AwardEngine(award_year=2025)
cost = engine.calculate_shift_cost(
    employee_id="EMP001",
    award_level=1,
    employment_type=EmploymentType.FULL_TIME,
    shift_date=date(2026, 3, 15),
    start_time=time(9, 0),
    end_time=time(17, 0),
)
print(f"Gross: ${cost.gross_pay}, Super: ${cost.super_contribution}")
```

Output (Sunday shift example):
- Gross Pay: $289.20 (8 hours @ $36.15)
- Superannuation: $22.17 (11.5% of $192.80)
- Total Employer Cost: $311.37
